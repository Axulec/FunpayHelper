Инструкция по запуску

Файлы:
- bot.py          : код бота (не содержит ваш токен)
- requirements.txt: зависимости
- .env            : файл окружения (вставь свой BOT_TOKEN)

Перед запуском:
1) Открой .env и впиши ваш токен:
   BOT_TOKEN=8453464342:AA... (пример)

2) Установи зависимости:
   pip install -r requirements.txt

3) Запусти бота:
   python bot.py

Рекомендация для постоянной работы (24/7):
- Залей репозиторий на GitHub и подключи к Render.com (бесплатный план), или используй VPS.

Команды/кнопки:
- "Да ✅" -> callback /gototask (кнопка). Команда /gototask показывает кнопку запуска.
- "Поднял ✅" -> /upproduct (можно нажать кнопку или использовать команду /upproduct)
- "Прекратить напоминание ❌" -> /stoptask (кнопка или команда)

Важное:
- Я не вставлял ваш BOT_TOKEN — вставьте его в .env перед запуском.
